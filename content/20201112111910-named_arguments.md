# Description

Allowes named arguments[^1] to be used to access parameters in random
order, as in [JavaScript](20200922162127-named_parameters).

# Syntax

``` php
function tralala(string $a, string $b, string $c)
{
    echo $a.' '.$b.' '.$c;
}

tralala(b: 'la', c: 'la', a: 'tra');
// tra la la
```
