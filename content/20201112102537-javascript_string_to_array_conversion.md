# Description

Strings can be [converted](20200613170756-javascript_type_conversion)
into [arrays](20200826201029-arrays) with the
[spread](20201014094144-spread) operator:

# Syntax

``` javascript
const chars = [..."abc"];
console.log(chars); // Array(3) [ "a", "b", "c" ]
```
