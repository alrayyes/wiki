# Description

TryInto[^1] is an attempted conversion that consumes self, which may or
may not be expensive.

# Delcaration

``` rust
pub trait TryInto<T> {
    type Error;
    fn try_into(self) -> Result<T, Self::Error>;
}
```

# Implementors

``` rust
impl<T, U> TryInto<U> for T
where
    U: TryFrom<T>,
```
