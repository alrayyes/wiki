# Origin

Latin for "mindful remembering"

# Description

Cache the return value of a function

# Example

``` javascript
const memo = {}

function fib(n) {
    if(memo[n]) {
        return memo[n]
    }
    if(n <= 1) {
        return 1
    }

    return memo[n] = fib(n -1) + fib(n - 2)
}

fib(5)
console.log(memo) // { '2': 2, '3': 3, '4': 5, '5': 8 }
```
