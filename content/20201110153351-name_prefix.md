# Introduction

This is an option to prefix the named of the routes imported in
configuration files.

# Syntax

``` yaml
app:
    resource: ../controller/routing.yaml

api:
    resource: ../controller/routing.yaml
    # this prefix is added to all the action route names
    name_prefix: api_
    # this prefix is added to all the action URLs
    prefix: /api
```
