# Operators

``` javascript
console.log(4 >= 3)
console.log(4 <= 3)
console.log(4 == 3)
console.log(4 != 3)
console.log(0 == false)
console.log(0 === false)
console.log(0 != false)
console.log(0 !== false)
```

The only value in JavaScript that is not equal to itself is NaN

``` javascript
console.log(NaN == NaN)
```
