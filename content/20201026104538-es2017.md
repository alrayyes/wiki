# Async functions

## Fulfilling a [promise](20200911154351-promises)

``` javascript
async function asyncFunc() {
  return 123;
}

asyncFunc().then((x) => console.log(x));
// 123
```

## Rejecting a [promise](20200911154351-promises)

``` javascript
async function asyncFunc() {
  throw new Error("Problem!");
}

asyncFunc().catch((err) => console.log(err));
// Error: Problem!
```
