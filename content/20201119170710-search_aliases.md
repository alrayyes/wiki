# Description

`#[doc(alias = "<alias>")]` can be specified on items to add search
aliases when searching through [rustdoc's](20201119165454-rustdoc) UI.
See the PR[^1] for more information.

# Syntax

``` rust
#[doc(alias = "bar")]
struct Foo;
```
