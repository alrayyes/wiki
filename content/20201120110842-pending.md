# Description

pending[^1] creates a future which never resovles, representing a
computation that never finishes.

# Declaration

``` rust
pub fn pending<T>() -> Pending<T>
```

# Notable traits

``` rust
impl<T> Future for Pending<T>
    type Output = T;
```

# Examples

``` rust
use core::future;

let future = future::pending();
let () = future.await;
unreachable!();
```
