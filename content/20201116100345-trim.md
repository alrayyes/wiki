# Description

Removes all
[whitespace](20201116100926-characters_that_count_as_whitespace) from
both ends of a [String](20200922164551-strings). Combination of
[trimStart](20201116100205-trimstart) and
[trimEnd](20201116100239-trimend)

# Syntax

``` javascript
console.log(' abc '.t)
```
