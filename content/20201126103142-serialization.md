# Origin

English series "in order"

# Defintion

The process of taking data from one format and converting it to a more
generic form that can be used by other programs.

# Example

``` javascript
const data = {
    hello: 'world'
}

JSON.stringify(data)
```

``` python
import json

dataFromJS = "{hello: 'world'}"

# parse
x = json.loads(dataFromJS)
```
