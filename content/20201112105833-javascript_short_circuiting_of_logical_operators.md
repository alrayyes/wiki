# Syntax

``` javascript
console.log(null || "user")
console.log("Agnes" || "user")
console.log(0 || -1)
console.log("" || "!?")
console.log(true || "X")
console.log(false || "X")
console.log(true && "X")
console.log(false && "X")
```
