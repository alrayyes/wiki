# Syntax

## Number

``` typescript
var num: number
num = 123
num = 123.24
num = '123' / Error
```

## String

``` typescript
var str: string

str = '123'
str = 123 // Error
```

## Boolean

``` typescript
var bool: boolean

bool = true
bool = false
bool = 'tralala' // Error
```
