# Description

Introduced in [PHP 8.0](20201109133834-php_8_0), class name literals[^1]
is a new alternative to `get_class()`

# Syntax

``` php
$foo = new Foo();
var_dump($foo:class)
```

[^1]: <https://wiki.php.net/rfc/class_name_literal_on_object>
