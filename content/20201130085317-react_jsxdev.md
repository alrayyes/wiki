# Description

Support development compiles for React 17's
[jsx](20201130085253-react_jsx) and
[jsxs](20201130084716-jsxs_react_factory_function) factory functions.
See PR[^1].

# Example

``` json
// ./src/tsconfig.dev.json
{
    "extends": "./tsconfig.json",
    "compilerOptions": {
        "jsx": "react-jsxdev"
    }
}
```
