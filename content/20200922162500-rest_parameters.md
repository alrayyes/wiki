# Introduction

Introduced in [ES6](20201030093404-es6)

# Syntax

``` javascript
function logAllArguments(...args) {
    for (const arg of args) {
        console.log(arg);
    }
}

logAllArguments(1, 2, 3)
```

``` javascript
function logAllArguments(pattern, ...args) {
    console.log(pattern)
    for (const arg of args) {
        console.log(arg);
    }
}

logAllArguments("asdf", 1, 2, 3)
```
