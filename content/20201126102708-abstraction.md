# Origin

Latin abstrahere "drawm from"

# Definition

Hide implementation details from the user. For exapmle by extending
classes.

# Examples

``` typescript
abstract class Fish {
    swim() { return '>>>>>' }
}

class Shark extends Fish  {
    name: 'shark'
}

class Tuna extends Fish {
    name: 'tuna'
}
```
