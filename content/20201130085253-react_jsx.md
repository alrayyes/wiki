# Description

Support production compiles[^1] for React 17's
[jsx](20201130085253-react_jsx) and
[jsxs](20201130084716-jsxs_react_factory_function) factory functions.

# Example

``` json
// ./src/tsconfig.json
{
    "compilerOptions": {
        "module": "esnext",
        "target": "es2015",
        "jsx": "react-jsx",
        "strict": true
    },
    "include": [
        "./**/*"
    ]
}
```
